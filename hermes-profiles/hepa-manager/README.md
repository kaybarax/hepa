# hepa-manager

Hermes profile for the HEPA engineering manager. This profile owns task clarity,
lane orchestration, review arbitration, human-friendly PR content, and the
manager-owned GitHub lifecycle.

Files:

- `SOUL.md` - stable manager identity and hard limits
- `config.example.yaml` - example Hermes settings; copy into the real profile
  config and replace placeholder paths

The manager writes the PR body for humans. HEPA validates that body, appends
deterministic evidence, and performs the safe mechanical publish step.
