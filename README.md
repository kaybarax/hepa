# HEPA

HEPA is an independent Rust-first sibling of HOCA. HOCA is used as a read-only
behavioral reference and parity-test source, not as a runtime dependency.

Hermes Kanban/dashboard is the default operator surface for HEPA v1.0.0, while
HEPA's deterministic registry, lane records, artifacts, and state machine remain
authoritative. CLI and headless operation must continue to work when Hermes is
unavailable.

This repository is the initial workspace skeleton for the v1.0.0 implementation.
